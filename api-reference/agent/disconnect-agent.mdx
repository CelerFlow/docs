---
title: "Disconnect agent"
description: "Gracefully disconnect an agent from its fleet."
api: "POST /disconnect-agent"
---

## Request

<ParamField body="agent_id" type="string" required>
  The agent UUID to disconnect.
</ParamField>

### Example

```bash
curl -X POST https://ristessocgeoiifdbukq.supabase.co/functions/v1/disconnect-agent \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"agent_id": "agent_001"}'
```

## Response

```json
{
  "success": true,
  "status": "disconnected"
}
```

<Note>
  Disconnecting preserves the agent's historical data (traces, health checks). Use [remove agent](/api-reference/agent/remove-agent) to permanently delete.
</Note>
