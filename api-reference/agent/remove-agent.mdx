---
title: "Remove agent"
description: "Permanently remove an agent and all its data."
api: "DELETE /remove-agent"
---

## Request

<ParamField body="agent_id" type="string" required>
  The agent UUID to remove.
</ParamField>

<Warning>
  This action is irreversible. All traces, health checks, and configuration for this agent will be permanently deleted.
</Warning>

### Example

```bash
curl -X DELETE https://ristessocgeoiifdbukq.supabase.co/functions/v1/remove-agent \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"agent_id": "agent_001"}'
```

## Response

```json
{
  "success": true,
  "deleted": true
}
```
