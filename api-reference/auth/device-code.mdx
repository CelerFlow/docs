---
title: "Request device code"
description: "Start the device code OAuth flow for CLI authentication."
api: "POST /cli"
---

## Request

```json
{
  "action": "device-code"
}
```

## Response

```json
{
  "device_code": "dc_abc123",
  "user_code": "ABCD-1234",
  "verification_url": "https://celerflow.vercel.app/cli-auth",
  "expires_in": 900,
  "interval": 5
}
```

| Field | Description |
|---|---|
| `device_code` | Used to poll for the token. |
| `user_code` | Displayed to the user. |
| `verification_url` | Where the user signs in and enters the code. |
| `expires_in` | Validity in seconds (15 minutes). |
| `interval` | Minimum seconds between poll attempts. |
