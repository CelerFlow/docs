---
title: "Poll device token"
description: "Poll for the access token after user authorization."
api: "POST /cli"
---

## Request

```json
{
  "action": "device-token",
  "device_code": "dc_abc123"
}
```

## Response (pending)

```json
{
  "status": "pending"
}
```

## Response (success)

```json
{
  "access_token": "eyJ...",
  "token_type": "bearer",
  "expires_in": 3600
}
```

<Note>
  Poll at the interval from the device code response (default: 5 seconds). Faster polling may trigger rate limiting.
</Note>
