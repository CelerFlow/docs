---
title: "Poll device token"
description: "Poll for the access token after the user has authorized the device code."
api: "POST /cli-device-token"
---

## Request

<ParamField body="device_code" type="string" required>
  The device code from the [request device code](/api-reference/auth/device-code) response.
</ParamField>

### Example

```bash
curl -X POST https://ristessocgeoiifdbukq.supabase.co/functions/v1/cli-device-token \
  -H "Content-Type: application/json" \
  -d '{"device_code": "dc_abc123"}'
```

## Response (pending)

```json
{
  "status": "pending"
}
```

## Response (success)

```json
{
  "access_token": "eyJ...",
  "token_type": "bearer",
  "expires_in": 3600
}
```

<Note>
  Poll at the interval specified in the device code response (default: 5 seconds). Polling faster may result in rate limiting.
</Note>
