---
title: "Generate setup code"
description: "Generate a one-time code for connecting an agent to a workspace."
api: "POST /agent"
---

## Request

```json
{
  "action": "generate-setup-code",
  "fleet_id": "uuid"
}
```

## Response

```json
{
  "code": "ABC123",
  "fleet_id": "uuid",
  "expires_at": "2026-03-28T07:15:00Z"
}
```

<Note>
  Setup codes are single-use and expire after 15 minutes.
</Note>
