---
title: "Billing usage"
description: "Retrieve usage data for billing and cost attribution."
api: "GET /billing-usage"
---

## Request

<ParamField query="fleet_id" type="string">
  Filter usage by fleet. Omit for account-wide usage.
</ParamField>

<ParamField query="start_date" type="string">
  Start date (ISO 8601). Default: beginning of current billing period.
</ParamField>

<ParamField query="end_date" type="string">
  End date (ISO 8601). Default: now.
</ParamField>

### Example

```bash
curl "https://ristessocgeoiifdbukq.supabase.co/functions/v1/billing-usage?fleet_id=fleet_abc123" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
