---
title: "Create fleet"
description: "Create a new fleet to organize your agents."
api: "POST /create-fleet"
---

## Request

<ParamField body="name" type="string" required>
  Fleet name. Must be unique within your account.
</ParamField>

### Example

```bash
curl -X POST https://ristessocgeoiifdbukq.supabase.co/functions/v1/create-fleet \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "Production"}'
```

## Response

```json
{
  "id": "fleet_abc123",
  "name": "Production",
  "created_at": "2026-01-15T10:30:00Z"
}
```
