---
title: "Create workspace"
description: "Create a new workspace (fleet) to organize agents."
api: "POST /fleet"
---

## Request

```json
{
  "action": "create",
  "name": "Production"
}
```

<ParamField body="name" type="string" required>
  Workspace name. Must be unique within your account (max 50 chars).
</ParamField>

## Response

```json
{
  "id": "uuid",
  "name": "Production",
  "is_default": false,
  "created_at": "2026-01-15T10:30:00Z"
}
```
