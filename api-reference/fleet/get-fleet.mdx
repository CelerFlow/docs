---
title: "Get fleet"
description: "Retrieve details for a specific fleet."
api: "GET /get-fleet"
---

## Request

<ParamField query="fleet_id" type="string" required>
  The fleet UUID.
</ParamField>

### Example

```bash
curl "https://ristessocgeoiifdbukq.supabase.co/functions/v1/get-fleet?fleet_id=fleet_abc123" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
