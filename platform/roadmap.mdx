## Coming soon

### MCP Gateway

Connect **any** MCP-compatible agent framework (LangGraph, CrewAI, OpenAI Agents SDK, Strands, Mastra) to CelerFlow without installing a plugin. The gateway federates multiple MCP servers behind a single endpoint with policy checks and tracing.

[Learn more →](/integration/mcp-gateway)

### Agent Payments

Give agents their own USDC wallets for paying MCP services that charge via the x402 / MPP protocol (HTTP 402). Budget controls include per-transaction limits, daily/monthly caps, recipient allowlists, and HITL thresholds for high-value transactions.

### Paid MCP Endpoints

Make your agents accept payments from other agents. Set a price per tool call, CelerFlow generates a paid MCP endpoint, and other agents pay in USDC to access your agent's capabilities.
