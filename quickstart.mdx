## Create your first fleet

<Steps>
  <Step title="Sign in">
    Go to [celerflow.vercel.app](https://celerflow.vercel.app) and sign in with Google.
  </Step>

  <Step title="Create a fleet">
    From the dashboard, click **New fleet**. Give it a name like "Production" or "Dev agents".

    A fleet is a logical group of agents. Use it to organize agents by project, team, or environment.
  </Step>

  <Step title="Install the CLI">
    Install `ocauthz` globally:

    ```bash
    npm install -g ocauthz
    ```

    Verify the installation:

    ```bash
    ocauthz --version
    ```
  </Step>

  <Step title="Log in via CLI">
    Authenticate with your Celerflow account:

    ```bash
    ocauthz login
    ```

    This opens your browser for Google OAuth. After authenticating, your credentials are saved locally to `~/.ocauthz/credentials.json`.
  </Step>

  <Step title="Connect an agent">
    Generate a setup code from the dashboard for your fleet, then connect:

    ```bash
    ocauthz connect --code <SETUP_CODE>
    ```

    Your agent is now registered and will begin reporting health to Celerflow.
  </Step>

  <Step title="Verify the connection">
    Check agent status from the CLI:

    ```bash
    ocauthz status
    ```

    Or visit the dashboard to see your agent appear in the fleet view with a green health indicator.
  </Step>
</Steps>
